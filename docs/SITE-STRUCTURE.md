### 5. About Page (`/about`) — **NEEDS EXPANSION**
**Purpose:** Author credentials, E-E-A-T signal, trust building  
**Schema:** `Person` with `sameAs`  
**Current status:** ⚠️ Exists but lacks credentials  

**Required content:**
- Full author name + professional photo
- Bio paragraph with credentials and background
- Expertise areas listed explicitly
- Social proof: "I run this blog and test every strategy on it"
- Links: LinkedIn, Twitter/X, GitHub (for `sameAs` schema)
- List of featured posts

**Required schema:**
```json
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "[Author Name]",
  "url": "https://[domain]/about",
  "image": "https://[domain]/images/author.jpg",
  "jobTitle": "Digital Marketing Strategist",
  "sameAs": [
    "https://linkedin.com/in/[handle]",
    "https://twitter.com/[handle]"
  ]
}
```

## Breadcrumb Design

Every blog post should display and encode breadcrumbs:
```
Home > Blog > [Category] > [Post Title]
```

BreadcrumbList schema:
```json
{
  "@type": "BreadcrumbList",
  "itemListElement": [
    { "@type": "ListItem", "position": 1, "name": "Home", "item": "/" },
    { "@type": "ListItem", "position": 2, "name": "Blog", "item": "/blog" },
    { "@type": "ListItem", "position": 3, "name": "[Post Title]", "item": "/blog/[slug]" }
  ]
}
```
